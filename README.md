## Proyectos destacados

### BuddyFinder - Proyecto Final de Ciclo (TFG)
Plataforma de adopción de perros construida en **4 módulos independientes** que se comunican a través de una API REST central.

| Módulo | Tecnología | Descripción |
|--------|-----------|-------------|
| [API REST](https://github.com/Andy-McFly/TFG_BuddyFinderAPI) | Spring Boot + MySQL | Núcleo del sistema con endpoints para perros, usuarios, refugios, solicitudes y favoritos |
| [Android](https://github.com/Andy-McFly/TFG_BuddyFinderAndroid) | Kotlin | App para usuarios: catálogo de perros, solicitudes de adopción, perfil |
| [Web](https://github.com/Andy-McFly/TFG_BuddyFinderWeb) | HTML · CSS · JS | Panel para refugios: gestión de perros, historial y solicitudes recibidas |
| [PC Admin](https://github.com/Andy-McFly/TFG_BuddyFinderPC) | Python · Tkinter | Herramienta de administrador: gestión de refugios y transferencias |

### [Fish Rampage](https://github.com/Andy-McFly/Juego_FishRampage)
Juego arcade 2D en Java con **programación multihilo**. Game loop, renderizado y enemigos en hilos separados con sincronización para evitar race conditions.
`Java` `Swing` `Multithreading`

### [Ghouls N' Dungeon](https://github.com/Andy-McFly/Juego_Ghouls-Dungeon)
RPG de combates por turnos con persistencia completa en MySQL. Sistema de fases, tienda, progresión de personaje y diario de partidas con historial de victorias y derrotas.
`Java` `MySQL` `Maven` `JDBC`

### [PasitosApp](https://github.com/Andy-McFly/App_PasitosApp)
App Android de rastreo de rutas en tiempo real sobre Google Maps. GPS, trazado de Polyline, lectura de batería con BatteryManager y persistencia SQLite con recuperación de ruta al reabrir la app.
`Java` `Google Maps SDK` `GPS` `SQLite`

### [Mis Pedidos](https://github.com/Andy-McFly/App_MisPedidos)
App Android de gestión de pedidos online con CRUD completo, consumiendo una API REST PHP via OkHttp. Dos entidades relacionadas (pedidos y tiendas) con RecyclerView y navegación por pestañas.
`Java` `Android` `OkHttp` `REST API`

### [Cliente FTP](https://github.com/Andy-McFly/Cliente_Servidor)
Cliente FTP de escritorio con interfaz gráfica Swing. Navega directorios, sube y descarga ficheros desde un servidor FTP real usando Apache Commons Net.
`Java` `Swing` `FTP` `Apache Commons Net`

### [Siete y Media](https://github.com/Andy-McFly/SieteYMedia)
Juego de cartas para hasta 4 jugadores en Java con patrón MVC. Mazo completo de 40 cartas, lógica de turnos, puntuación acumulada por rondas y gestión de empates.
`Java` `AWT` `MVC`
